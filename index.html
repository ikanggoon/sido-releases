Features

생산성을 높여주는 핵심 기능들

✓

할 일 목록

카테고리별 그룹, 완료/숨기기, 드래그 정렬로 할 일을 체계적으로 관리하세요.

▦

Priority Matrix

아이젠하워 4분면으로 긴급도와 중요도를 한눈에 파악하고 우선순위를 정하세요.

◼

위젯 모드

항상 위에 고정, 투명도 조절, 폰트 크기 토글. 작업 중에도 할 일을 확인하세요.

📁

카테고리 관리

추가, 삭제, 이름변경, 숨기기, 정렬. 크로스 카테고리 드래그앤드롭으로 빠르게 이동.

🎨

5가지 테마

라이트, 다크, 블루, 그린, 로즈. 취향에 맞는 테마로 기분 좋게 관리하세요.

🔄

자동 업데이트

새 버전이 나오면 앱에서 바로 알림. 한 번의 클릭으로 최신 버전을 유지하세요.

📄

데이터 관리

JSON 내보내기/가져오기로 데이터를 안전하게 백업하고 복원하세요.

📅

마감일 관리

할 일별 마감일 설정, 기한 초과 시 경고 표시로 놓치는 일이 없도록.

⌨

키보드 단축키

Ctrl+N 새 할일, Ctrl+B 사이드바, Ctrl+E 내보내기. 마우스 없이도 빠르게.

♿

접근성

ARIA 속성, 포커스 관리, WCAG AA 색상 대비. 모든 사용자를 위한 배려.
